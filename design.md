# Sinapta Design System

## Palette
- Background: `#080B14` (near-black, deep navy)
- Surface: `#0D1120` (card bg)
- Surface2: `#111827` (elevated)
- Primary: `#0A84FF` (electric blue)
- Accent: `#00E5BE` (cyan/turquoise)
- Text: `#F0F4FF` (near white)
- Muted: `#8892A4` (grey)
- Border: `rgba(255,255,255,0.08)`

## Typography
- Display/Titles: `Syne` (Google Fonts) — bold, geometric, high-impact
- Body: `DM Sans` — clean, readable
- H1: 72px / 4.5rem, weight 800
- H2: 48px / 3rem, weight 700
- H3: 28px / 1.75rem, weight 600
- Body: 16px / 1rem, 1.7 line-height
- Caption: 13px / 0.8125rem
- Label: 11px / 0.6875rem, uppercase, letter-spacing 0.1em

## Motion
- Scroll reveal: fade + translateY(24px) → 0, 600ms, ease-out, stagger 100ms
- Hover buttons: scale 1.02, shadow increase, 200ms
- Hover cards: translateY(-4px), shadow increase, 200ms
- Header: transparent → blur + bg on scroll, 300ms ease-in-out

## Layout
- 12-col grid, 24px gap, 80px side margins desktop, 40px tablet, 20px mobile
- Section padding: 120px vertical desktop, 80px tablet, 60px mobile
- 8px spacing scale

## Style Direction
- Dark premium, like ElevenLabs / Linear
- Gradient mesh backgrounds (blue/cyan, very subtle)
- Grid line textures (low opacity)
- Glowing CTA buttons
- Clean geometric cards with subtle borders
